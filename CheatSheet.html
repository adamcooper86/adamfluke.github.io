Ruby Enumerables Cheat Sheet

An enumerable is a type of Ruby object that has included the enumberable module. They must have a method call each defined for it, and the enumberable module uses that method to complete the methods described below. Not all enumberable classes have the same functions, so check RubyDocs for specific examples.

All? Any? Collect/Map Count Detect Each_Slice Each_With_Index Inject Select

All?

Returns true if everything in the block returns true.

    
      ["cat","bat","mat"].all?{|word| word.length == 3} #returns true
      ["cat","bat","hats"].all?{|word| word.length == 3} #returns false

Any?

Returns true if anything in the block returns true.

    
      ["cat","bat","mat","zipper"].any?{|word| word.length == 6} #returns true
      ["cat","bat","hats"].any?{|word| word.length == 6} #returns false

Collect / Map

Returns a new array with the result of the running block, once for every element.(See also .map)(Both map and collect have destructive versions !)

    
      ["cat","bat","mat"].collect{|word| word * 2}
        #returns ["catcat", "batbat", "matmat"]
      [1,2,3,4,5].map!{|integer| integer * 5}
        #returns [5,10,15,20,25]

Count

Count counts how many times an object is in the enumberable, or give it a block and it will tell you how many times the block returns true.

    
      ["cat","bat","mat"].count{|word| word.length == 3} #returns 3
      ["cat","bat","hats"].count{|word| word.length == 3} #returns 2
      ["cat","bat","hats"].count{|word| word == "bat"} #returns 1

Detect

Detect returns the first object that is found to be true and if it is not found it returns nil.

    
      ["cats","bat","mat"].detect{|word| word.length == 3} #returns bat
      ["cat","bat","hats"].detect{|word| word.length == 5} #returns nil

Each_Slice

Itterate the given block, it slices up the enumerable and passes the slices into the block. It passes an array of values with a length = to the arguement.

    
      return_array = []
      ["cat","bat","mat","jack","rat","hummus"].each_slice(2){|words| return_array << (words[0] + words[1])}
       #returns nil, but adds values to return_array["catbat","matjack","rathummus"]

Each_With_Index

Each_with_index itterates over an enumerable, but passes in a count parameter that increments by one each time.

    
      array = []
      ["cat","bat","mat"].each_with_index{|word, count| word.length + count}
        #array = [3,4,5]
      ["cat","bat","hats"].each_with_index{|word, count| array << word << count}
        #array = [cat,0,bat,1,hats,2]

Inject

    
      ["cat","bat","mat"].inject(:+)
        #returns "catbatmat"
      ["cat","bat","hats"].inject(1){|sum, string| sum * string.length}
        #returns 27
      ["cat","bat","hats"].inject do |memo, word|
        memo.length > word.length ? memo : word
      end
      #returns hats

Select

Returns an array containing all elements of the enumerator for which the block returns a true value.

    
      ["cat","bats","mat","rats"].select{|word| word.length == 4}
        #array = ["bats","rats"]
      ["cat","bat","hats"].select{|word| word == "bat"}
        #array = ["bat"]
